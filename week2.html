Week 2 | Computer-aided Design Assignment

The assignment for the week was to create an enclosure for an Arduino Uno or Raspberry Pi (raster, vector, 2D, 3D, render, animate, simulate, ...) and document it on my Class Page using Fusion360. I included pictures and original files on my page. I mentioned what worked and did not work when creating the enclosure.
